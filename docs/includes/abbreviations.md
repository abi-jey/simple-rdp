*[RDP]: Remote Desktop Protocol - Microsoft's proprietary protocol for remote desktop access
*[NLA]: Network Level Authentication - Security feature requiring authentication before full RDP connection
*[CredSSP]: Credential Security Support Provider - Security protocol for NLA authentication
*[PDU]: Protocol Data Unit - A unit of data in the RDP protocol
*[RLE]: Run-Length Encoding - Compression algorithm used for bitmap data
*[MCS]: Multipoint Communication Service - T.125 protocol layer used by RDP
*[GDI]: Graphics Device Interface - Windows graphics subsystem
*[PIL]: Python Imaging Library (Pillow) - Python library for image processing
*[FPS]: Frames Per Second - Measure of screen capture rate
*[MCP]: Model Context Protocol - Open protocol for LLM tool integration
*[LLM]: Large Language Model - AI model like GPT or Claude
*[NTLM]: NT LAN Manager - Windows authentication protocol
*[TLS]: Transport Layer Security - Encryption protocol for secure communication
*[ASN.1]: Abstract Syntax Notation One - Standard for data serialization
